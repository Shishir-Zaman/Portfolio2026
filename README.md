# 2026 Personal Portfolio — Shishir Zaman

A high-end, minimalist luxury portfolio designed to showcase premium brand identity, visual design, and motion graphics. Built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**, featuring a dynamic topology animation and a seamless, high-performance UI.

![Portfolio Preview](https://res.cloudinary.com/dw10neaqr/image/upload/q_auto/f_auto/v1778832522/ChatGPT_Image_May_15_2026_02_07_20_PM_yu5gsg.png)

## 💎 Design Philosophy
This portfolio is built on the principle of **Minimalist Luxury**. It avoids clutter in favor of bold typography, smooth glassmorphism, and interactive micro-animations. The primary teal accent (`#00f5ff`) provides a high-energy focal point against a deep obsidian backdrop.

## 🚀 Key Features
- **Topology Animation:** Custom-built Canvas API background with organic, flowing lines and mouse interaction.
- **Dynamic Text Looping:** Rotating professional titles across the "About" section for a living identity.
- **Project Deep Dives:** Detailed project pages with immersive galleries and strategic breakdown sections.
- **Single Source of Truth:** All content (Socials, Projects, Services) is managed through a centralized `data/content.ts` file for effortless updates.
- **Glassmorphism UI:** Premium frosted-glass components with real-time backdrop blurring.
- **Fully Responsive:** Optimized for everything from mobile devices to ultra-wide 4K monitors.

## 🛠️ Tech Stack
- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/) & Custom SVGs
- **Deployment:** [Vercel](https://vercel.com/)

## 📦 Getting Started

### Prerequisites
- Node.js 18.17 or later
- npm or pnpm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Shishir-Zaman/Portfolio2026.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📝 Configuration
To update your name, bio, projects, or social links, simply edit:
`app/data/content.ts`
